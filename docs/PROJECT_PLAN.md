# Project Plan

## Game Overview
This document outlines the development of a Stardew Valley-style game. The game features farming, crafting, and interactions with NPCs in a 2D environment.

## Scene Architecture
The game consists of various scenes including the player’s farm, the world map, and UI overlays for inventory and interactions.

## Core Systems Design
Key systems include a GameManager to manage game state, a TimeSystem to control in-game time, and an InventorySystem to manage items.

## Directory Structure
The project's directory structure is outlined below:

```
scenes/
    player/

    farm/

```

## Development Roadmap
- **Phase 1**: Basic game mechanics and player movement.
- **Phase 2**: Implement farming and crop systems.
- **Phase 3**: Create NPCs and interaction systems.
- **Phase 4**: Develop UI and inventory management.
- **Phase 5**: Polish and test the game before release.
